# Icon Specifications

**Material Symbols**

- Weight: `300`
- Grade: `-25`
- Optical Size: `24px`
- Rounded

**Ionicons**

- We use `color-wand`, `flask-filled`, `logo-github`, `logo-google-playstore`, `repeat`, and `shuffle`.

**Custom Icons**

- `nothing-arrow-right`
- `repeat-one` is a custom icon based on Ionicons' `repeat` & `calendar-number` icons.
